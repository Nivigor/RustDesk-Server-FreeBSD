# RustDesk-Server-FreeBSD
 Install your own [open source RustDesk Server](https://github.com/rustdesk/rustdesk-server) on FreeBSD: 
 1. Create a folder for server and go to it.
 2. Run ```fetch https://github.com/Nivigor/RustDesk-Server-FreeBSD/releases/latest/download/rustdeskserver.tgz```
 3. Run ```tar xzf rustdeskserver.tgz```
 4. Run ```./install.sh```
